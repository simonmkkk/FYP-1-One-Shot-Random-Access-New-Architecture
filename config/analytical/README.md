# Analytical Config Subdirectory Documentation

Analytical model configuration subdirectory that stores analytical calculation configuration files for each figure.

**File List:**
- `figure1.yaml` - Figure 1 configuration (shared with Figure 2), defines parameters such as N value list, M range, etc.
- `figure345.yaml` - Figure 3-5 combined configuration, defines parameters such as M, I_max, N ranges, etc.
